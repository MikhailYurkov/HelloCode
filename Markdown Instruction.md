# Инструкция для работы с Markdown

## Выделение текста
<br/><br/>
Чтобы выделить текст курсивом необходимо обрамить его звездочками {\*\}.
Например, *вот так*.


Чтобы выделить текст полужирным, необходимо оьрамить его двойными звездочками (\**\) Например, **вот так**
